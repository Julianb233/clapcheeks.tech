# Clap Cheeks Mobile

React Native (Expo) mobile app for Clap Cheeks dating intelligence platform.

## Setup

```bash
cd mobile
npm install
npx expo start
```

## Environment

Copy `.env.example` to `.env` and fill in your Supabase credentials:

```bash
cp .env.example .env
```

## Development

- `npm start` — Start Expo dev server
- `npm run ios` — Start on iOS simulator
- `npm run android` — Start on Android emulator
