# Milestone 4: Analytics & AI Coaching (v0.4)

Dashboard, conversion tracking, and AI coaching engine.

## Goal

Turn the anonymized metrics flowing into the cloud into actionable insights for users. Build a web dashboard showing swipe-to-date conversion funnels, AI-powered coaching tips, conversation suggestions, and automated weekly performance reports.

## Phases

| Phase | Name | Description |
|-------|------|-------------|
| 16 | Analytics dashboard | Swipes, matches, conversations, dates booked, spending, and conversion rates |
| 17 | AI coaching engine | Claude API analyzes user patterns and generates personalized improvement tips |
| 18 | Conversation AI | AI reply suggestions for dating app chats (beyond iMessage) |
| 19 | Weekly reports | Automated PDF/email performance summaries with AI recommendations |

## Key Metrics Tracked

- Swipe-to-match rate (by app, by day, by profile type)
- Match-to-conversation rate
- Conversation-to-date rate
- Date-to-second-date rate
- Cost per date (from spending tracker)
- Best performing hours, opener styles, and conversation patterns

## Dependencies

Milestone 3 (Dating App Automation) must be complete before beginning Milestone 4.
Analytics depend on data flowing from the local agent (Milestone 2).
