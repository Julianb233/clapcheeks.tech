```
Mac chat.db ──► match_sync ──► clapcheeks_matches
              ─► backfill   ──► clapcheeks_messages
```

## Phases

1. **Inventory** — count current roster rows, unenriched profiles, stale stages
2. **Ingest texts** — pull chat.db via SSH, backfill `clapcheeks_messages` per match
3. **Enrich profiles** — for each match, run `match_profile_engine` (bio / IG / zodiac / DISC)
4. **Recompute** — run `roster/health.py` for metrics + stage reclassification
5. **Verify** — live check at `/matches` + spot-check 3 profiles end-to-end
