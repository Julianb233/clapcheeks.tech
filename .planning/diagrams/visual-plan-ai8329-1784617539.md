# AI-8329 Phase 44: Autonomy Engine — Visual Plan

## ASCII Architecture

```
 swipe UI / capture ──record──> convex.autonomy.recordSwipeDecision ──> [swipe_decisions]
                                                                              │
 cron / on-demand ──train──> _trainPreferenceModel ──(lib/preference-model)──┤
                                                                              ▼
                                                                     [swipe_preferences]  (weights,bias,accuracy)
 candidate profile ──features──> predictSwipe ─(lib/preference-model.predict)─> {prob, confidence}
                                     │
                                     ▼
 auto-swipe loop ──> autoSwipeDecision ─(lib/auto-swipe.decideSwipe)─> like|pass|skip|STOP
                       (rate-limit window from swipe_decisions count)

 AI reply / touch ──> routeProposedAction ─(lib/approval-gate.routeAction)─> send | queue → [approval_queue]
                       (reads [autonomy_config] level + confidence)

 stale scan ──> staleRecovery ─(lib/stale-recovery)─> re-engage candidates
 dashboard  ──> confidenceDashboard ─(lib/confidence-dashboard)─> buckets, accuracy, auto/queued ratio
```

## Component table
| Component | Purpose | Inputs | Outputs | Deps |
|---|---|---|---|---|
| lib/autonomy/preference-model | AUTO-01 learn/predict swipe | decisions[], features | model, prob, confidence, accuracy | none (pure) |
| lib/autonomy/auto-swipe | AUTO-02 decide swipe w/ rate limits + low-conf stop | prediction, rate state, thresholds | like/pass/skip/stop | preference-model |
| lib/autonomy/approval-gate | AUTO-05 route by level+confidence | level, confidence, overrides | send/queue/drop | none |
| lib/autonomy/stale-recovery | AUTO-04 find stale convos | conversations[], now | re-engage candidates | none |
| lib/autonomy/confidence-dashboard | AUTO-06 aggregate stats | approvals, predictions, model | dashboard summary | none |
| convex/autonomy.ts | wrappers | Convex ctx | queries/mutations | lib + new tables |
| schema: swipe_decisions, swipe_preferences | persistence | - | - | - |
