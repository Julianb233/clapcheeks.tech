  Breakdown dimensions: emotional, communication, physical, summary text.
  </behavior>
  <implementation>
  Use element-based scoring from research:
  - ELEMENT_SCORES matrix: Fire-Fire=80, Fire-Earth=40, Fire-Air=85, Fire-Water=35, Earth-Earth=75, Earth-Air=45, Earth-Water=80, Air-Air=80, Air-Water=50, Water-Water=85
  - MODALITY_MODIFIER: Same=-5, Cardinal+Mutable=+5, all others=0
  - overall = clamp(elementBase + modalityMod, 0, 100)
  - emotional = overall * (water present ? 1.1 : 0.9), clamped 0-100
  - communication = elementBase * (same modality ? 0.85 : 1.0), clamped 0-100
  - physical = elementBase * (fire present ? 1.15 : 0.9), clamped 0-100
  - summary = text based on overall score range (0-30 Challenging, 31-50 Tricky, 51-70 Moderate, 71-85 Strong, 86-100 Excellent)

  Export getCompatibility function and CompatibilityScore interface.
  </implementation>
